# Tomato Disease Detection using EfficientNetB0

This repository contains the full code implementation and training outputs for a **Case Study Report** on *Tomato Disease Detection* using Deep Learning, designed for **NNDL (CSE23433)**.

## 🎯 Objective
The primary objective of this project is to develop a highly accurate Convolutional Neural Network (CNN) capable of identifying and classifying three localized plant diseases from the **PlantVillage** dataset:
1. `Tomato_Bacterial_spot`
2. `Tomato_Early_blight`
3. `Tomato_Late_blight`

*The dataset was specifically capped to 1,000 images per class to establish perfect class balancing.*

## ⚙️ Architecture (EfficientNetB0)
The core model relies on an incredibly robust transfer-learning approach using **EfficientNetB0** (pretrained on ImageNet). 

A custom top-class probability node was created to refine predictions:
- Global Average Pooling 2D
- Custom Dense Layers (256 → Dropout 0.4 → 128 → Dropout 0.3)
- Final Softmax Output Layer (3 Classes)

**Training approach:** It utilized a **Weighted Focal Loss** algorithm rather than Categorical Crossentropy. Focal Loss aggressively directs the network to focus computational weight on hard-to-classify diseased anomalies instead of standard background leaves.

## 📊 Evaluation & Results
The model was tested heavily on an unseen validation holdout slice across a stringent two-step phase process (Phase 1 frozen base head tuning, Phase 2 deep-layer unfreezing). 

* **Final Test Accuracy:** `96.44%`
* **Test Loss Bound:** `0.0224`
